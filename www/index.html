RefLek przeglądarka leków refundowanych

RefLek to bardzo prosty program, służący do przeglądania aktualnych leków refundowanych, ogłoszonych przez Ministra Zdrowia.

Prosta wyszukiwarka w programie jest zbudowana tak, aby uprościć maksymalnie proces szukania leku - wystarczy wpisać dowolną część nazwy, dawkowania, bądź dowolnego innego pola, a program pokaże wszystkie leki, które spełniają kryteria wyszukiwania

Po wybraniu danego leku w tabelce, można przeczytać dokładne informacje na temat leku.

Program jest darmowy i działa na wszystkich popularnych systemach operacyjnych!

Wersja dla windows

Pobierz reflek.exe, zapisz go na dysku i uruchom. Program nie wymaga instalacji, można go trzymać na dysku USB, bądź w dowolnym folderze. Nie wymaga także uprawnień administratora.

Wersja windows

Wersja dla Mac/Linux

Sćiagnij reflek.jar i zapisz go w dowolnym miejscu. W systemie Mac OS X uruchom go klikając na nim, w systemie linux uruchom z konsoli używając komendy java -jar reflek.jar
